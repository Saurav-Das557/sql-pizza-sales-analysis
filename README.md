# 🍕 SQL Pizza Sales Analysis  

## 📌 Project Overview  
This project analyzes a pizza sales dataset using SQL to uncover business insights.  
Through queries on **orders, pizzas, pizza types, and order details**, we answer basic, intermediate, and advanced analytical questions related to sales, revenue, and customer preferences.  

---

## 📂 Dataset  
The project uses the following CSV files:  
- `orders.csv` – order IDs, dates, and times  
- `order_details.csv` – pizzas included in each order  
- `pizzas.csv` – pizza IDs, sizes, and prices  
- `pizza_types.csv` – pizza categories and names

## 🚀 How to Use  
1. Import the CSV files into your SQL environment (MySQL, PostgreSQL, SQLite, etc.).  
2. Create the tables (`orders`, `order_details`, `pizzas`, `pizza_types`).  
3. Run the queries listed in `Questions.txt`.  
4. Compare results with the provided screenshots for validation.

## 📌 Insights  
- The **Greek Pizza** is the highest priced pizza.  
- The **Large size** is the most commonly ordered.  
- The **Classic category** contributes the most in terms of both **quantity** and **revenue**.  
- Average pizzas ordered per day: **138**.  
- Total revenue generated: **$817,860.05**.  
- Total orders placed: **21,350**.
